# XML-теги поддерживаемых форматов

## BPMN 2.0

- [OMG BPMN 2.0.2](https://www.omg.org/spec/BPMN/2.0.2/) — спецификация формата, включая XML-модель элементов `definitions`, `process`, `task`, `sequenceFlow`, `gateway`.
- [BPMN 2.0 XML Schema](https://www.omg.org/spec/BPMN/20100501/BPMN20.xsd) — XSD-схема для проверки XML-тегов и атрибутов BPMN.

## draw.io / diagrams.net

- [Manually edit the XML source of your draw.io diagram](https://www.drawio.com/docs/manual/advanced/diagram-source-edit/) — описание XML-источника диаграммы и корневого тега `mxGraphModel`.
- [Generate and validate draw.io diagrams with AI](https://www.drawio.com/docs/reference/diagram-generation/) — справочные материалы diagrams.net для создания и проверки `.drawio` XML.

## Archi / ArchiMate

- [The Open Group ArchiMate Model Exchange File Format](https://www.opengroup.org/open-group-archimate-model-exchange-file-format) — описание обменного XML-формата ArchiMate.
- [ArchiMate 3.0 XML Schema](https://www.opengroup.org/xsd/archimate/3.0/) — XSD-схемы тегов `model`, `element`, `relationship` и связанных атрибутов.
