# Идеи улучшения сервиса

- Добавить полноценный draw.io renderer через совместимый viewer diagrams.net.
- Добавить ArchiMate Exchange renderer на базе `elkjs` для автоматической раскладки.
- Поддержать Mermaid и PlantUML как дополнительные текстовые форматы схем.
- Добавить вкладки ошибок: XML-ошибки, ошибки формата, ошибки рендеринга.
- Сохранять историю последних XML в `localStorage`.
- Добавить переключение тем редактора и диаграммы.
- Добавить миникарту диаграммы.
- Добавить drag-and-drop загрузку XML-файлов.
- Добавить сжатие URL-параметра для больших схем.
- Добавить визуальные тесты примеров для GitHub Actions.
