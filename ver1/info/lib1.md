# Рассмотренные JS-библиотеки

## BPMN

- `bpmn-js`: зрелая библиотека просмотра и редактирования BPMN 2.0. Выбрана для текущего viewer, потому что умеет импортировать BPMN XML и строить полноценную диаграмму в браузере.
- `diagram-js`: низкоуровневый движок, на котором построен `bpmn-js`. Использовать напрямую имеет смысл только для собственных нотаций.

## XML-редактор

- `CodeMirror 5`: браузерный редактор с подсветкой XML, поиском, заменой и клавиатурными командами. Выбран, потому что легко работает в статической странице GitHub Pages.
- Monaco Editor: функциональный редактор уровня VS Code, но тяжелее для простой статической страницы.
- Ace Editor: легкий редактор с подсветкой XML, но готовый поиск/замена и настройка уступают CodeMirror для этой задачи.

## draw.io

- `mxGraph`: исходная библиотека draw.io, но проект устаревший и сложнее подключается как современная зависимость.
- diagrams.net embed/viewer: практичный путь для полноценного просмотра draw.io через внешний viewer.
- В текущей версии реализовано универсальное SVG-представление `mxCell`-элементов, чтобы сервис оставался автономным.

## Archi / ArchiMate

- Специализированных стабильных browser-only JS viewer для файлов Archi меньше, чем для BPMN.
- Возможные направления: парсинг ArchiMate Exchange XML, построение графа через `Cytoscape.js`, `D3.js` или `elkjs`.
- В текущей версии Archi отображается универсальной SVG-картой элементов и связей.

## Экспорт и визуализация

- SVG создается средствами браузера.
- PNG создается через отрисовку SVG в `canvas`.
- PDF создается через печать браузера, что совместимо с GitHub Pages без серверной части.
