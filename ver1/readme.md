# xml-dia ver1

Клиентское приложение для просмотра XML-диаграмм в браузере. Основной режим работы
рассчитан на GitHub Pages, без серверной части и без Node.js.

### run

- Основной запуск на GitHub Pages через корень проекта:
  https://bpmbpm.github.io/xml-dia/
- Прямой запуск текущей версии на GitHub Pages:
  https://bpmbpm.github.io/xml-dia/ver1/index.html
- Запуск сохраненной схемы через URL: откройте приложение, загрузите или вставьте XML,
  нажмите `Сохранить url`, затем используйте полученную ссылку с параметром `?xml=`.
- Локальный запуск после скачивания репозитория через статический HTTP-сервер:
  из корня проекта выполните `python3 -m http.server 8000` и откройте
  http://127.0.0.1:8000/ver1/index.html или http://127.0.0.1:8000/.
- Локальный запуск без сервера: откройте файл `ver1/index.html` в браузере.
  В этом режиме браузер может блокировать автоматическую загрузку примеров из папки
  `dia`, поэтому используйте кнопку `Открыть XML`, вставку XML в редактор или ссылку
  с параметром `?xml=`.
