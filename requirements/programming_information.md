# requirements

## code
Код пиши на Браузерном JS. Не используй node даже в целях тестирования.  

### Pull request
В Pull request указывай измененные файлы. При отсутсвии изменений явно указывай, что изменений нет.  

### doc
Коммениируй код на русском. Файлы описания и докуменацию, Pull request пиши на русском. 

## run
Запуск предполагается на github pages (основной режим), а также при скачивании на локальном компьютере. При неообходимости в index.html может быть предусмотрен вручную устанавливаемый режим запуска программы.
